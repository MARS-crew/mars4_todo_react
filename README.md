# Todo Project

### Project Start

```
1. git clone

2. cd mars4_todo_react

3. npm install or yarn add

4. npm run dev
```

### Directory Structure

```js
root
└─src // app, index, cookie
  ├─ api // auth, workbooks
  ├─ assets // color, font..
  ├─ components // components file
  ├─ page // page file
  ├─ store // redux
  ├─ style // global style
  ├─ utils // cookie, hooks
  ├─ views // Views file
  ├─ App.jsx
  └─ main.jsx
```

### using library

-   React 18
-   Styled-components
-   redux, redux-toolkit
